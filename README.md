# Mess-and-jar
Hobby educational implementation of web-based messenger, starting with a public room and ability for "unlimited" amount of members participating.

## Tech stack
### Frontend
Remix
### Backend (server)
Golang server

#### Checklist
- [x] Set your own username
- [x] Change user

- [x] Send messages to others
